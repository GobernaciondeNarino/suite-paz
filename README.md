# Suite PAZ

**Plugin de WordPress** para la **Gobernación de Nariño** que publica los datos del proyecto de **Paz** ("Así llegó la paz a Nariño") como **gráficos, mapas coropléticos y módulos** insertables en cualquier página mediante **shortcodes**, con un **editor de datos** en el panel de administración.

Recreación del plugin [`tic-suite`](https://github.com/GobernaciondeNarino/tic-suite) adaptada al proyecto de Paz.

> 🚧 En desarrollo en la rama `actualizar-paz`. El diseño está en `docs/superpowers/specs/`.
